# [GeoMapSSH](https://github.com/egladman/GeoMapSSH)

## Prerequistes

### Install Dependencies
```
nvm use
npm install
```

## How to use

### Quickstart
```
npm start
```

### Advanced
```
npm start -- -l docs/example -p 3030
npm start -- --log docs/example --port 3030
```

Open `localhost:3030` in a browser
