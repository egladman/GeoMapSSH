## Prerequistes

### Install Dependencies

Run the following commands:
```
nvm use
npm install
```

## Quick Start

### Basic

Launch app

```
npm start
```

### Advanced

Lint

```
npm run lint
```

Create test file

```
npm test
```

Launch app

```
npm run start-dev
npm start -- --log docs/example --port 3030
```

*Note:* When overwriting the default port it's currently necessary to update the value in `public/index.html`
