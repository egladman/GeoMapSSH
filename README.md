## Prerequistes

### Install Dependencies

Run the following commands:
```
nvm use
npm install
```

## How to use

### Simple

Run the following command:
```
npm start
```

### Advanced

```
npm start -- -l docs/example -p 3030
npm start -- --log docs/example --port 3030
```
